# Vita Enterprise Pattern Layer

Use this directory with `../DESIGN.md`, `../tokens/vita.tokens.json`, `../styles/vita-open-design.css`, and `../reference/apex-reference.json` when generating enterprise application prototypes.

## Files

| File | Purpose |
| --- | --- |
| `page-templates.json` | Vita page shell, 12-column grid, page template, dialog, and drawer guidance. |
| `enterprise-patterns.json` | Enterprise page blueprints for dashboards, reports, editable grids, master-detail pages, transaction forms, approvals, and wizards. |
| `component-state-matrix.json` | Component variants and interaction states for buttons, forms, regions, reports, grids, filters, badges, overlays, timelines, and wizards. |

## Source Boundary

The canonical URLs in these files point to `oracleapex.com`. The evidence was captured from the fallback Universal Theme host after switching Theme Style to Vita and verifying `apex-theme-vita` plus primary token `#056ac8`.

Do not use these files as a cross-theme style system. They are Vita-only.
