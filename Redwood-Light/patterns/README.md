# Redwood Light Enterprise Pattern Layer

Use this directory with `../DESIGN.md`, `../tokens/redwood-light.tokens.json`, `../styles/redwood-light-open-design.css`, and `../reference/apex-reference.json` when generating enterprise application prototypes.

| File | Purpose |
| --- | --- |
| `page-templates.json` | Page shell, grid, page-template, Dialog, Drawer, and inline overlay guidance. |
| `enterprise-patterns.json` | Enterprise page blueprints for dashboards, reports, grids, master-detail pages, transaction forms, approvals, and wizards. |
| `component-state-matrix.json` | Component variants and states for buttons, forms, reports, filters, badges, overlays, timelines, and wizards. |

These files are Redwood Light-only. Canonical URLs stay on `oracleapex.com`; `oracleapex.cn` is recorded only as fallback evidence.
