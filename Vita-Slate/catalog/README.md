# Vita - Slate Machine-Readable Catalogs

Use these files when a prototyping tool needs structured lookup tables instead of parsing Markdown tables.

| File | Purpose |
| --- | --- |
| `component-catalog.json` | Standard component catalog derived from the Vita - Slate `DESIGN.md` component table. |
| `navigation-icon-catalog.json` | Navigation classes, navigation entries, Font APEX families, icon classes, and observed icon categories. |
| `open-design-readiness.json` | Readiness audit that lists the companion artifacts needed for enterprise Open Design generation. |

These files are Vita - Slate-only. The canonical URLs remain on `oracleapex.com`; `oracleapex.cn` is recorded only as fallback evidence.
